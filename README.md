# Password Strength Checker

A cybersecurity tool built in Python that analyzes 
password security and helps users create stronger passwords.

## Features
- Analyzes password strength
- Gives a score out of 100
- Estimates crack time
- Checks against common password databases
- Suggests improvements

## How to Run
git clone https://github.com/kainat167/password-checker
cd password-checker
python3 password_checker.py

## Technologies Used
- Python 3
- Regular Expressions (re module)
